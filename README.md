🧮 Python Calculator

📌 Description

This is a simple calculator project developed using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division. This project is created as part of learning Python programming and improving problem-solving skills.

🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- User-friendly input

🛠️ Technologies Used

- Python

🎯 Purpose

This project helps beginners understand:

- Basic Python syntax
- Conditional statements
- User input handling

▶️ How to Run

1. Install Python
2. Run the file using:
   python calculator.py
